# NBA Machine Learning Regression Analysis

## Project Overview
This project analyzes NBA player performance using multiple linear regression.

The objective is to understand which offensive statistics are most related to points per game.

## Dataset
The analysis uses NBA player statistics from the 2024/25 regular season.

## Methodology
- Data cleaning
- Exploratory analysis
- Multiple linear regression
- Model interpretation
- Residual analysis

## Tools
- Excel
- Linear Regression
- Data Analysis Toolpak
- Statistics

## Key Results
The model showed a strong relationship between points per game and variables such as field goal attempts and field goal percentage.

## What I learned
This project helped me understand how regression models can be used to explain real-world data and support data-driven conclusions.
